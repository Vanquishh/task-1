HTML & CSS Basics Summry

Html Summary

HTML (HyperTextMarkup Lagnguage) is used to define our page content,structure and meaning. You don't use it for styling purpose. Use CSS for that insted!

  • HTML uses "elements" to descibe (annotate) content
  • HTML typically have an opening tag, and then a closing tag
  • HTML use "elements" to describe (annotate) content You can also have voiid(empty) like images
  • You can also configure elements with attributes
  • There's a long list of available elements but you'll gain experience over the time,no worries.

Learn more about all available HTML elements on the MDN HTML element referance

CSS Summary

CSS(Cascading Style Sheets) is used for styling our page content.

  • Styles are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code dupllication, you tipically use global styles (e.g. via the "style" element) Insted
  • Alternatively, you can work with external stylesheet files which you "link" to
  • When working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood.

Learn more about all available CSS properties and values on the MDN CSS property reference
